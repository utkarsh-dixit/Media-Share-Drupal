INTRODUCTION
------------

Media Share module allows to share images on a node. This module
allows the user to share images on facebook, google plus,
pininterest and twitter.

REQUIREMENTS
------------

No special requirements 

INSTALLATION
------------
 
 * Install as you would normally install a contributed Drupal module. See:
   https://www.drupal.org/docs/8/extending-drupal-8/installing-contributed-modules-find-import-enable-configure-drupal-8
   for further information.

CONFIGURATION
-------------

 * Customize the MediaShare settings in Administration » Configuration »
   MediaShare configuration settings.
